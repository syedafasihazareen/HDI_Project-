Human Development Index Prediction System

This project predicts the Human Development Index (HDI) using Artificial Intelligence. Users enter:

  • Life Expectancy
  • Mean Years of Schooling
  • Expected Years of Schooling
  • GNI Per Capita

Objective

To predict the HDI score quickly using a Machine Learning model instead of manual calculations.

Technologies Used

  • Python
  • Flask
  • HTML
  • CSS
  • Scikit-Learn
  • Pandas
  • Joblib

Machine Learning Model

Random Forest Regressor
